About me

I have always been in love with IT. I remember using computers and gadgets since I have use of reason. The first computer I used was a desktop HP Pavilion a367c with Windows XP and only 512MB of RAM. I enjoyed tinkering with it so much...


Some games like Super Mario 64 and TLoZ: The Wind Waker moved me so much that led me to wonder how could I make such a great game like those too. Then I started with the game dev world at the age of 11 with Game Maker, and made some tiny projects...


Since then I was sure this would be the passion I wanted to dedicate myself to.


Now I often read about the latest news, put new ideas into practice and propose myself challenging projects that help me to become a better developer. I am also always open to new creative ideas from my friends and relatives. I usually program with Javascript, but I have also programmed with other languages that include C#, Java, Python and PHP. I am enthusiastic about functional programming and I think it should be important to teach that paradigm in schools too.
